# Защита существующего контента

## Принцип: «Не навреди»

Существующий утверждённый текст ГДД — результат дизайн-решений. Claude НЕ имеет права менять смысл без явного запроса.

## Абсолютные инварианты

Эти утверждения ИСТИННЫ всегда. Если твой текст им противоречит — ты ошибся.

### Мир и фракции
- В 1ORBIT ОДНА фракция. Нет компаний, рас, сторон
- PvP = конфликт пилотов внутри одной фракции
- Социальная идентичность = кланы, отряды, личная репутация
- Все карты доступны всем (ограничение = уровень, не фракция)

### NPC
- NPC НЕ имеют щита (SH, ShieldAbs, Shield). Только HP
- Каждый NPC = три варианта: Normal, Hyper, Ultra
- Каждый NPC = одна модель поведения (BehaviorModel): Drifter, Swarm, Herder, Sentinel

### Валюты
- Ровно три валюты: Кредиты (Cr), Нокс (N), SOL
- SOL → N: односторонняя конвертация
- Cr ↔ N: конвертация НЕВОЗМОЖНА

### Монетизация
- Бесплатный игрок получает ПОЛНЫЙ геймплей
- Премиум = скорость прогресса, НЕ боевые преимущества
- Запрещено: P2W, скрытые шансы лут-боксов, агрессивные попапы

### Баланс
- ГДД описывает ПОВЕДЕНИЕ. Числа хранятся в таблице балансировки
- Один код = один параметр = одно значение во всех документах

## Процедура при редактировании

1. Прочитай ВЕСЬ текущий текст страницы
2. Выдели ТОЛЬКО те фрагменты, которые нужно изменить
3. Для каждого изменения покажи: Было → Стало → Причина
4. НЕ трогай остальной текст
5. Жди подтверждения

## Типичные ошибки Claude

- Добавление щита NPC (Shield/SH) — ЗАПРЕЩЕНО
- Введение новых валют или фракций — ЗАПРЕЩЕНО
- «Улучшение» формулировок при редактировании — ЗАПРЕЩЕНО
- Добавление правил «по аналогии с другими играми» — ЗАПРЕЩЕНО
